# ゲーム概要
「水位調整」×「世界反転」をコンセプトにした、クリック操作のみで進んでいく3Dアクションパズルゲーム。

ここには __”現実世界”__ と __”死後の世界”__ があり、水面を介して行き来することができる。
不思議なネックレスの力を手にした少女は、2つの世界を行き来して、逃げた飼い猫を追いかけているうちに、自分の隠された過去を少しずつ取り戻す。
同時に彼女は、世界の真相に足を踏み入れることになるーーーー
